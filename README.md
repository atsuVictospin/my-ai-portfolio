# MNIST_001
このコードはMNIST画像分類AIの学習用コードです.
PyTorchの基本的な使い方を習い、簡単なCNNモデルを構築しています。
結果も可視化して具体的に結果がわかるようにしています。

environment.yml は、誰でも
```
conda env create -f environment.yml
```
で環境を再現できます。
